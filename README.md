# okex-py
OKEx数字货币自动交易python语言SDK (非官方)  
OKEx Cryptocurrency Exchange python SDK (Unofficial)

本项目基于V5 API

## 使用例子 Example
``` python3
import okex.v5.account_api as account
import okex.v5.market_api as market
result = accountAPI.get_positions()

print(result)
```
